⚙️ Google Sheets Setup

Required for all devices to sync data.

One-time setup:
1. Open your Google Sheet → Extensions → Apps Script
2. Replace all code with the new apps_script.gs (v2)
3. Deploy → New deployment → Web App
4. Execute as: Me · Who has access: Anyone
5. Copy the Web App URL and paste below
6. Sheet sharing → Anyone with link → Viewer
⚠️ If you updated the Apps Script code, make sure to create a New Deployment (not update existing) to get a new URL.
